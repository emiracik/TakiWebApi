===============================================
EN ÇOK KULLANILACAK ENDPOINT'LER
===============================================

🔐 AUTHENTICATION:
POST /api/auth/register - Kayıt ol
POST /api/auth/login - Giriş yap
PUT /api/auth/change-password - Şifre değiştir

👤 KULLANICI PROFİLİ:
GET /api/users/{id} - Kullanıcı bilgileri
PUT /api/users/{id} - Profil güncelle

📍 ADRESLERİM:
GET /api/useraddresses/user/{userId} - Adreslerim
POST /api/useraddresses - Adres ekle
PUT /api/useraddresses/user/{userId}/setdefault/{addressId} - Varsayılan adres
DELETE /api/useraddresses/{id} - Adres sil

💳 KARTLARIM:
GET /api/usercreditcards/user/{userId} - Kartlarım
POST /api/usercreditcards - Kart ekle
DELETE /api/usercreditcards/{id} - Kart sil

💰 CÜZDANIM:
GET /api/wallet/{userId} - Bakiye
GET /api/wallet/{userId}/transactions - İşlem geçmişi
POST /api/wallet/{userId}/deposit - Para yükle
POST /api/wallet/{userId}/withdraw - Para çek

🚕 YOLCULUKLARIM:
GET /api/trips/passenger/{passengerId} - Yolculuk geçmişi
GET /api/trips/passenger/{passengerId}/total-cost - Toplam harcama
POST /api/trips - Yeni yolculuk

🔔 BİLDİRİMLERİM:
GET /api/notifications/user/{userId}/unread - Okunmamış bildirimler
GET /api/notifications/user/{userId}/unread/count - Bildirim sayısı
PUT /api/notifications/{id}/mark-as-read - Okundu işaretle
PUT /api/notifications/user/{userId}/mark-all-as-read - Tümünü okundu yap

⭐ DEĞERLENDİRMELER:
GET /api/DriverRatings/driver/{driverId}/average - Şoför puanı
POST /api/DriverRatings - Şoför değerlendir
GET /api/UserRatings/user/{userId}/average - Kullanıcı puanı

===============================================
NOTLAR:
===============================================
- Tüm endpoint'ler Authorization gerektirir (JWT Token)
- POST/PUT işlemlerinde JSON body gönderilir
- Pagination için pageNumber ve pageSize parametreleri kullanılır
- Tarih formatı: YYYY-MM-DD veya ISO 8601
- ID'ler integer tipindedir
- Soft delete kullanılır (IsDeleted = true)
