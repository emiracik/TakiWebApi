===========================================
TESTING NOTES
===========================================

1. Make sure TaksiWebApi is running before testing
2. Database connection should be successful
3. All endpoints return JSON data
4. Use tools like Postman, curl, or VS Code REST Client
5. Check Console output for any errors
6. Pagination starts from pageNumber=1
7. Search terms are case-insensitive
8. Date format: YYYY-MM-DD
